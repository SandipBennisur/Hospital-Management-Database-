# Hospital Management Database

A SQL-based Hospital Management Database project developed using MySQL.

## Features
- Department Management
- Doctor Management
- Patient Management
- Appointment Management
- Admission Management
- Billing Management

## Database Tables
- department
- doctors
- patient
- appointment
- admission
- billing

## Files
- create_tables.sql
- insert_data.sql
- analytics_queries.sql

## Technologies
- MySQL
- SQL

## How to Run
1. Execute `create_tables.sql`
2. Execute `insert_data.sql`
3. Execute `analytics_queries.sql`
